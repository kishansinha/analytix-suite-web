MOD-02 How to Use
📋 What This Module Does

Progress Tracking takes results from two to five assessments and visualises how student performance has changed over time. It produces individual student progress charts as well as class-wide trend charts across AOs, topics, components, and overall performance. A focused tracking option lets you zero in on specific skills or topics across the test series. Results include an Excel summary of key statistics.

📁 Files You Need
1
Two to five Data Files

One Analytix_Data_Template_File.xlsx per test, filled in with student scores for that assessment. Download the template from the Analytix Suite home page (Downloads tab).

ℹ️

The Data Template is available in the Downloads tab on the Analytix Suite home page. Each test file should follow the same column structure — same AO codes and topic names — so progress can be tracked consistently.

ℹ️

No AOs File needed. Unlike some other modules, Progress Tracking reads AO and topic information directly from the Data Files — you do not need to upload a separate AOs File.

🔢 Step-by-Step
1
Upload your test files (Panel 1)

Drag and drop or click to upload between 2 and 5 Data Files. Files are accepted in any order — you will assign a date and a label to each one after uploading.

2
Assign dates and labels

For each uploaded file, enter the test date and a short label (e.g. Test 1, Mock Exam, Final). Labels appear on the time axis of all progress charts. Dates are used to order the tests chronologically.

3
Load tests

Click Load tests. The module reads all files, detects shared students, AOs, and topics, and unlocks Panel 2. A summary pill for each test appears showing student count and question count.

4
Configure your analysis (Panel 2)

Enter the grade or class name. Choose whether to include individual student charts. Optionally enable Focused AO / Topic tracking to select specific AOs or topics for deep-dive charts.

5
Generate Progress Report

Click Generate Progress Report. Panel 3 shows a live log as the server processes your data. Run time is typically 20–60 seconds depending on class size and number of tests.

6
Download your results (Panel 3)

Click Download ZIP. The ZIP contains all charts as PNG files organised into subfolders (class charts, individual student charts, focused tracking charts) plus an Excel summary report.

💡

You can adjust Font size in the Fine-tuning section if chart labels appear too small or too large for your class size.

🔍 Focused AO / Topic Tracking

When enabled, this option generates additional deep-dive charts for the specific AOs and topics you select. Useful when you want to closely monitor progress in a particular skill area (e.g. AO3 — evaluation) or topic (e.g. kinematics) across the test series.

⚠️

AO and topic selectors are only populated after tests are loaded. Load your files first, then enable focused tracking and make your selections.

📊 What You Get
📈
Learning Curves

Line charts showing how overall performance, AO scores, topic scores, and component scores have changed across the test series — for the class and for individual students.

📦
Boxplots

Show the distribution of scores at each time point — median, spread, and outliers — across AOs, topics, and overall performance. Useful for seeing whether the class is converging or diverging over time.

🎻
Violin Plots

Show the full shape of score distributions across AOs and topics at each time point. Reveals whether performance is clustered, bimodal, or spread widely within a skill area.

🌡️
Heatmaps

Display how each student has performed across AOs and topics over the test series. Makes it easy to spot persistent weaknesses or meaningful improvements at a glance.

👤
Individual Student Charts

Progress charts, AO breakdowns, component scores, and a summary card for each student — shareable directly as personalised progress feedback.

🔍
Focused Tracking Charts

Deep-dive charts for selected AOs and topics — showing evolution of scores over time for both the class and individual students in those specific areas.

📋
Excel Summary Report

A formatted spreadsheet containing key statistics across all tests — class means, standard deviations, and per-student progress data by AO and topic.

💡 Tips
💡

Consistent student names: A student's name must be spelled identically across all test files for their progress to be tracked correctly. Check for typos before uploading.

💡

Absent students: Enter AB in the first data cell below the student's name in that test's file. They will be excluded from that test's data without breaking the overall progress chart.

💡

Unattempted questions: Enter NA — not 0. This preserves the distinction between a question not attempted and one incorrectly answered.

ℹ️

Your data is never stored. Files are processed on the server and deleted immediately after your ZIP is generated.
