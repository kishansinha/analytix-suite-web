MOD-00 How to Use
📋 What This Module Does

Descriptive Statistics computes a comprehensive set of statistical quantities for your selected variables — helping you understand the shape, spread, and behaviour of your data before running further analysis. Results are written to an Excel file that includes the computed metrics, supporting charts, plain-language interpretations, and recommendations for which comparative, correlation, and effect size tests are appropriate for your data.

ℹ️

No AOs File needed. This module requires only your Analytix Data File.

📁 File You Need
1
One Data File — Analytix_Data_Template_File.xlsx

Your filled-in Analytix Data Template with student scores. Download the template from the Downloads tab on the Analytix Suite home page if you haven't already.

🔢 Step-by-Step
1
Upload your Data File (Panel 1)

Drag and drop or click to upload your Analytix_Data_Template_File.xlsx. Click Load the data file. The module reads the file and unlocks Panel 2.

2
Select variables (Panel 2)

Choose which variables to analyse. Variables are grouped into Overall score, Main AOs (e.g. AO1, AO2, AO3), Sub-AOs (e.g. AO2j, AO3d), and Topics. Use the quick-select buttons to pick a whole group, or tick variables individually.

3
Set analysis options (Panel 2)

Choose which statistical quantities to compute. Enable Generate interpretations to include plain-language explanations of each metric. Enable Enhanced plots (Seaborn) for higher-quality chart styling.

4
Confirm & Run

Click Confirm & Run. Panel 3 shows a live log as the analysis runs. Typical run time is 10–30 seconds depending on the number of variables selected.

5
Download your results (Panel 3)

Click Download ZIP. The ZIP contains an Excel report with all metrics, charts, interpretations, and test recommendations.

📐 About Variables

Once your file is loaded, Panel 2 displays all available variables derived from your data, grouped into four types:

🎯
Overall

Total percentage score across all questions. A good starting point for understanding the general performance distribution of the class.

📂
Main AOs

Scores grouped by top-level Assessment Objective category (e.g. AO1, AO2, AO3). Useful for understanding broad skill-level differences across the class.

🔬
Sub-AOs

Scores for individual AO codes (e.g. AO2j, AO3d). Enables granular analysis of specific skills.

📚
Topics

Scores grouped by topic label (e.g. kinematics, forces). Useful for identifying topic-level strengths and weaknesses.

💡

You do not need to analyse all variables. Select only those relevant to your current research question to keep the output focused and readable.

📊 What You Get
📈
Descriptive Metrics

Mean, median, mode, standard deviation, variance, IQR, skewness, kurtosis, confidence intervals, and outlier flags — computed per selected variable.

🔔
Normality Assessment

Statistical tests indicating whether each variable's scores are normally distributed — a key factor in choosing the right follow-up tests.

📉
Charts

Distribution plots, box plots, and other visualisations for each selected variable, embedded directly in the Excel output.

📝
Interpretations

When enabled, plain-language explanations of what each metric value means in context — no statistics background required to read them.

🧭
Test Recommendations

Based on the computed metrics, the module recommends which comparative tests (e.g. t-test, Mann-Whitney U), correlation tests, and effect size measures are appropriate for each variable — ready to use in Modules 3 and 4.

💡 Tips
💡

Run this module first. Descriptive Statistics is designed to be run before Comparative Analysis and Correlation Analysis. The test recommendations it generates tell you which tests to select in those modules.

💡

Interpretations are optional but recommended if you are new to statistics or plan to share the output with colleagues who are.

ℹ️

Your data is never stored. Files are processed on the server and deleted immediately after your ZIP is generated.
