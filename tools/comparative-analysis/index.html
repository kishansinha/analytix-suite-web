MOD-03 How to Use
📌 What This Module Does

Module 3 compares student performance across groups or time points. Upload 2 or more test files, choose your study design, select the variables to compare, and the module runs the appropriate statistical tests automatically — returning an Excel report and charts as a ZIP.

📋 Study Designs
⚖️
Control vs Experimental

Two independent groups (different students). Uses independent t-test and Mann-Whitney U. Upload exactly 2 files.

🔄
Pre–Post Comparison

Same students at two time points. Uses paired t-test and Wilcoxon signed-rank. Students are matched by name across files.

📊
Multiple Groups

Three or more independent groups. Uses one-way ANOVA and Kruskal-Wallis H with automatic Tukey HSD post-hoc. Upload 3+ files.

📈
Longitudinal Study

Same students at 3+ time points. Uses Repeated Measures ANOVA (with sphericity correction) and Friedman test. Upload 3+ files.

🚀 Step-by-Step
1
Choose your study design

Select one of the four cards. This determines which tests are shown and how many files are required.

2
Upload your files

Drop 2+ Excel files (one per group or time point). Files must follow the standard Analytix format.

3
Load & detect variables

Click "Load & Detect Variables". The module finds variables common across all files.

4
Name your groups and select variables

Edit the group labels and tick the variables to compare. Use the quick-select buttons to filter by type.

5
Choose tests and run

Sensible defaults are pre-checked for your design. Set α and click Run. Download the ZIP when ready.

📦 Excel Output
📋
Overview

Study design, group sizes, and which variables produced significant results.

📊
Results

One row per variable. Test statistics, p-values (colour-coded by significance), effect sizes, and CIs.

✅
Assumption Checks

Shapiro-Wilk normality and Levene's variance test per variable. Colour-coded pass/fail.

🔬
Post-Hoc

Pairwise comparisons for 3+ group designs. Exploratory post-hoc is run even when the main test is not significant — clearly flagged.

📐
Raw Descriptives

n, mean, SD, and median per group for every variable.

💡 Tips & Cautions
💡

Match student names exactly for paired designs. Pre–Post and Longitudinal designs match students by name. A student missing from any file is excluded from paired tests.

💡

Absent students are automatically excluded. Any student with an AB marker in any file is removed from the analysis entirely. For paired designs (Pre–Post, Longitudinal), a student absent in any one time point is excluded from all time points to keep the paired structure intact. Excluded students are listed by name in the Overview sheet of the Excel report.

💡

Set the Control/Pre role for 2-group designs. After uploading two files, use the role dropdown next to each file chip to designate which file is Control (or Pre) and which is Experimental (or Post). This determines the direction of the mean difference — a positive value means Experimental scored higher than Control.

💡

Check assumption results before interpreting tests. If Shapiro-Wilk flags non-normality, prefer the non-parametric result for that variable.

💡

Post-hoc tests always run for 3+ groups — even when ANOVA/Kruskal-Wallis is not significant. These are flagged as "exploratory" in the Post-Hoc sheet.

ℹ️

Effect sizes (Cohen's d, Hedges' g) are only available for 2-group designs. Glass's Δ is only available for Control vs Experimental (not Pre–Post, where there is no separate control group SD). For 3+ groups, η² and ε² are reported instead.

ℹ️

Your data is never stored. Files are processed on the server and deleted immediately after your ZIP is generated.
